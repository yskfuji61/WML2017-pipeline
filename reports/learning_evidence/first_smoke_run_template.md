# First smoke run evidence template (v4)

## Allowed wording

- local smoke run completed (when real WMH2017_ROOT evidence exists)
- local validation Dice or equivalent metric computed
- public WMH2017 local PoC only

## Prohibited wording

- official benchmark equivalent
- clinical / customer / production readiness
- SOTA
- READY_FOR_RELEASE

## Artifacts

- `run_context.json`
- `runtime_fingerprint.json`
- `prediction_manifest.json`
- `metrics_summary.json`
- `evidence_summary.md`
