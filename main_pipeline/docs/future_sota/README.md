# Future SOTA-candidate materials

These files are intentionally separated from the current smoke-pipeline working set.

Allowed use now:
- read as a future planning reference,
- extract open questions,
- list required evidence.

Forbidden use now:
- claim SOTA,
- claim leaderboard compatibility,
- claim winner reproduction,
- skip EXP-000,
- tune on test data,
- present to customers as validated performance.

Required gate before use:
1. EXP-000 data integrity evidence exists.
2. Source/license registry review is complete for the specific source.
3. Metric parity boundary is documented.
4. Human reviewer approves the next experiment.
